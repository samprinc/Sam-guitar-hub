# 🎸 Sam's Guitar Hub: 30-Day Learning Tracker

A zero-friction, single-page application designed to track, guide, and accelerate a 30-day acoustic (box) guitar learning journey. Built to eliminate the "what should I practice today?" excuse and keep the focus on actual execution. 

## 🚀 Features

* **📅 Interactive 30-Day Tracker:** Visual grid to mark days as done or skipped. Tracks current streaks and calculates overall completion percentage.
* **⏱ Built-in Metronome:** Web Audio API metronome integrated directly into the navigation bar so you don't have to open another app or switch tabs.
* **🗺 Phased Roadmap:** A curated, day-by-day curriculum divided into 4 phases, taking you from holding the guitar to playing your first full song.
* **🎵 Interactive Chord Library:** Essential beginner and intermediate chord shapes with ASCII visual diagrams and practical fingering tips.
* **🤚 Technique Guides:** Step-by-step instructions on posture, fretting clean notes, strumming, and reading guitar tabs.
* **📚 Curated Resources & Local Gear Guide:** Hand-picked apps, YouTube channels, and a specific gear guide tailored for buying and maintaining an acoustic guitar in Kenya.
* **💾 Data Persistence & Backup:** Automatically saves progress to browser `localStorage`. Includes 1-click JSON Export/Import functionality to back up your progress or move it across devices.

## 🛠 Tech Stack

This project was intentionally built to be lightweight, fast, and completely serverless.

* **Frontend:** HTML5, CSS3, JavaScript (ES6)
* **Styling:** Tailwind CSS (via CDN) + Custom CSS
* **Fonts:** Google Fonts (Space Grotesk, Inter)
* **Storage:** Browser `localStorage`

## ⚙️ How to Run

Because this is a zero-dependency, single-file application, there is no build step or server required.

1. Clone or download the repository.
2. Double-click the `index.html` file to open it in any modern web browser.
3. Start practicing. 

## 🔒 Data Privacy

All data (streaks, practice notes, and completed days) is stored locally in your browser. No data is sent to external servers. To keep your data safe if you clear your browser cache, use the **↓ Backup Data** button on the Tracker page.
